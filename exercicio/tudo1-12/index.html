todas as tag

A tag H1 serve para colocar titulo.
A tag H2 serve como subtitulo.
A tag P serve para iniciar um paragrafo.
A tag HR serve para fazer uma linha na horizontal.
A tag BR serve para quebrar uma linha.
A tag MARK serve para marca palavras.
A tag STRONG serve para deixar as palavras em negrito.
A tag EM serve para deixar texto em italico.
A tag IMG serve para colocar imagens.
A tag SMALL serve para deixar o texto piqueno.
A tag DEL serve para riscar uma palavra no meio.
A tag INS serve para riscar em baixo da palavra.
A tag CODE serve para deixar todas as letras com mesmo espaçamento.
A tag PRE serve para organizar espaço.
A tag ABBR serve para dar significado para abreviação.
A tag BDO:L serve para deixar a palavra de frente para trás.
A tag BDO:R serve para deixar a palavras de trás para frente.
A tag OL serve para numera uma lista.
A tag UL serve para colocar pontos em lista.
A tag LI serve para fazer a lista.
A tag OL+TYPE serve pra escolher numeros para lista.
A tag OL+START serve para escolher o inicio da contagem.
A tag LU+TYPE serve pra escolher qual tipo de bolinha.
A tag &.L.T; serve para colocar o sinal de menor (<).
A tag &.g.t; serve para colocar o sinal de maior (>).
A tag &#X=1F612 serve para colocar emoji.
A tag LINK:FAVICON serve para adicionar um favicon no seu site.
A tag A serve para adicionar um link na paralvra.
A tag A=TARGET serve para voce falar como quer abrir esse link.
